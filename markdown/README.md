## Code

### Inline Code
You can print to the terminal usign the `puts: "hello world` command.

### Multiline Code

#### Without highlighting

```
def hello world
    puts "Hello world"
end
```

#### With highlighting

```rb
def hello world
    puts "Hello world"
end
```
